Commands

Cooldown

Adds a delay between commands being activated. Helps to prevent spam.

Note that this is for all commands, not just one command repeating.

Repeat Delay

The amount of time that must pass until the next repeatable command is sent to chat.

Repeat Protection

The amount of non bot messages for the next repeatable message to go through.

Chat

Health Reminder

Get reminders to stretch, drink water, and take a break.

Log Messages

Logs chat messages to a text file.

Points

Initial Value

The amount of points a new user starts with.

Rate

The amount of points earned per 15 minutes.

Name

The name of your currency.

Moderation

Chat Filter

Filters some bad words from the chat. No filter is perfect, this is not a substitute for a mod. Triggering the filter will cause the user to be timed out for 5 minutes.

Enabled / Disabled

Reset to Default Save Edit Database

Edit Database

You can manually edit the database if the bot is having problems. You should only edit these files if you know what you are doing.

The file names must remain the same. This is a NeDB (MongoDB) database. You should research it before editing these files.

Copy the files in the folder before editing them. There is no way to reverse these changes once made.
